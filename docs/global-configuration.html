Global configuration

The Atlan constructor receives an optional globalConfig object that allows you to set global hooks, a global custom error handler, and a few more options.

FieldDescription
hooksHooks that will be applied to all models that don't have their own hooks object.
errorHandlerAn error handler that will be applied to all models that don't have their own errorHandler.
parseRequestAsJsonA boolean (default: true) that sets whether Atlan should parse requests as application/json. Uses body-parser.
